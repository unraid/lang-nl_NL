Hide Deprecated Applications=Verouderde applicaties verbergen
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=Deze instelling verbergt verouderde applicaties uit de lijst van beschikbare apps  Verouderde apps worden nog wel getoond in de geïnstalleerde en vorige apps secties
Hide Incompatible Applications=Incompatibele applicaties verbergen
This setting will hide applications that are listed as being incompatible with your version of Unraid=Deze instelling verbergt applicaties die niet verenigbaar zijn met deze versie van Unraid
Allow install of second instance=Installatie van tweede instantie toestaan
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=Deze optie toont een andere toets op kaarten en pop-ups, die het mogelijk maakt om reeds geïnstalleerde applicaties te herinstalleren met een standaardsjabloon  Dez optie kan soms verwarrend zijn, en wordt in het algemeen alleen gebruikt om te installeren wanneer een applicatie twee keer gelijktijdig draait
Show descriptions on application cards=Beschrijvingen op applicatie-kaarten weergeven
This option dictates whether the descriptions appear on the cards or not=Deze optie bepaalt of de beschrijvingen op de kaarten verschijnen of niet
Allow CA to check for updates to applications=CA toestaan te controleren op updates voor applicaties
This setting enables CA to check for updates to applications Note that Community Applications will always check for updates to itself=Met deze instelling kan CA controleren op updates voor applicaties. Merk op dat Community Applicaties altijd controleert op updates voor zichzelf
Allow CA to send any emergency notifications=Noodmeldingen versturen door CA
This setting allows CA to periodically scan your system and alert you to the presence of=Deze instelling laat CA regelmatig het systeem scannen en geeft een alarm bij de aanwezigheid van
1 Known malware and security vulnerabilities within applications=1 Bekende malware en beveiligingskwetsbaarheden binnen applicaties
2 Critical security vulnerabilities within Unraid OS=2 Kritieke beveiligingskwetsbaarheden binnen het Unraid besturingssysteem
3 Any installed applications which may seriously hinder the operation of your server=3 Enig geïnstalleerde applicatie welke de operatie van de server kan hinderen
Save CA debugging information=CA debug-informatie opslaan
Only enable this when advised to from the author of Community Applications=Schakel dit alleen in als dit wordt geadviseerd door de auteur van Community Applications
Enable developer mode=Ontwikkelaarsmodus inschakelen
This unlocks some extra features within CA generally only used by developers Not recommended to enable=Dit ontgrendelt enkele extra functies binnen CA die over het algemeen alleen door ontwikkelaars worden gebruikt. Niet aanbevolen om in te schakelen
